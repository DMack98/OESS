Current Features

  • self service virtual circuit provisioning web portal
  • integrated per port / vlan usage monitoring
  • sub-second provisioning
  • automatic failover and restoration to and from backup path
  • work groups for shared managment of resources
  • admin web interface for service management
  • switch and topology discovery
  • IDCP based inter-domain circuit provisioning
  • PefSONAR data export

Known Issues

There are several features we are unable to support with OpenFlow 1.0, we anticipate that these will be supported when we transition to OpenFlow 1.3.

  • MAC Address Learning
  • Q-in-Q
  • QoS (CoS)

Editing of multi-point circuits with a large number of endpoints may fail. Please contact the Internet2 NOC if you need to modify a large multi-point circuit and encounter an issue
